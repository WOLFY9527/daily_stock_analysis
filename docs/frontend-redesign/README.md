# WolfyStock Frontend UI/UX Redesign

## Index of Redesign Specifications

This directory contains the comprehensive frontend UI/UX redesign specifications for the WolfyStock system, drawing inspiration from the Cohere enterprise design language and adapting it for a professional financial intelligence platform.

### Design Goals
- Transition from a fragmented dark industrial UI to a unified, enterprise control-plane experience.
- Maintain a serious, premium, precise, and institutional feel.
- Strictly enforce a bilingual terminology system devoid of mixed or literal translations.
